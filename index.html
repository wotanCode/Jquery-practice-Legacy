¿Que es esto?

Bienvenido Pedro o visitante no deseado. Este es un sitio web personal de Pedro Yanez en el cual el ha recopìlado informacion y codigo de utilidad con el fin de ser usado en sitios y paginas web. En otras palabras, esto no debe estar en la nube si no que en el mejor caso debera ser una intranet, por otro lado al estar disponible los archivos fuentes, sera totalmente aceptable revisarlos y estudiarlos, de hecho para este proposito ha sido creado este sitio, con el fin de que sea una fuente de informacion privada.

Hay que aclarar ademas que si bien es una referencia, no se colocara o explicara el codigo basico desde principio a fin de HTML y cada cosilla que se vea por alli, al menos no en la version 1.0 Si no mas bien, almacenar codigo importante, recordatirios de relevancia y codigos dificiles. Disfrutad de esto.

Version 1.0

En esta version la plantilla se ha diseñado bajo codigo html5 y css3, ademas se uso el codigo GNU/GPL llamado CSS reset para que el navegador sea visto por igual independientemente de su ejecucion en las diferentes plataformas y/o navegadores. En el codigo fuente estan comentadas las partes consideradas de relevancia

Los CSS

Los efectos especiales de cada CSS estan realizados con sus ejemplos estan en un css especifico cada uno en caso de ser necesario. No los agrupo todo en styles porque entonces generaria un codigo excesivamente grande.

Notas

Se recomienda ver la tipografia en el css styles seccion de etiquetas ya que estan diseñadas de una manera especial, en los comentarios.
