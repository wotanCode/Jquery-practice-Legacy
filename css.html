Transformaciones

Aqui se agregan diferentes tipos de transformaciones CSS con una imagen mia. Esto se puede hacer tanto con imagenes, como con background de imagenes css. NOTA: Todas las imagenes son fondos background en css

La sintaxis basica para hacer transformaciones es: transform:function(valor) donde function es el tipo de transformacion que aplique. Ademas se debe usar el codigo propietario que son: -webkit- Chrome y Safari -moz- Mozilla -o- Opera -ms- Microsoft ver codigo fuente.

Transform Rotate

Las transformaciones en rotacion tienen diferentes tipos de medidas. Las medidas son

  1. deg (grados)
  2. grad (gradianes)
  3. rad (radianes)
  4. turn (giros)


Transform translate

Las transformaciones de traslado se usan con una sintaxis diferente: transform: translate(x,y) o bien cuando queremos que solo admita un valor seria asi: transform: translateX(valor)... el tipo de mededida puede ser en px, % es decir las medidas usuales.


Transform scale

Sirve para hacer el objeto mas grande... Lo escala, la sintaxis es: transform:scale(x,y)


Transform skew

Sesga la imagen al estilo photoshop, utiliza los mismos tipos de medida que que el rotate, es decir deg grad rad y turn... La sintaxis es: transform:skew(x,y)


Origen de transformacion

Por defecto el origen de las transformaciones es el centro y de alli hace los giros y todo lo demas pero este centro se puede cambiar con la siguiente sintaxis: transform-origin: valor valor. Ej transform-origin: right bottom; Ej2 transform-origin: center 50px;

Transformaciones multiples

Para ello se usa esta sintaxis. "transform: funcion(valor) funcion(valor) funcion(valor);"

Transformaciones interactivas

Para que las transformaciones sean interactivas y se activen cuando se ponga el puntero sobre ellas o bien cuando sea touch y pase el dedo encima se usan los seudoselectores :hover y :focus... Esto quiere decir que no se coloca en los parametros del css si no mas bien en el nombre del identificador o clase, es decir un ejemplo seria #traslaciones:hover{}

Transiciones

Base de Transiciones

Son los diferentes metodos usados para hacer una transicion a una transformacion en tiempos mas agradables y no de golpe con el :hover... Ver comentarios del codigo Css. Se hace si bien con el hover se usa ademas los comandos css -webkit-transition-property: all; el cual le dice de una vez que todos los comandos css3 se aplicaran cuando el puntero este arriba.. ej: -webkit-transition-property: -webkit-transition-; y -webkit-transition-duration: 3s; en el cual definimos tiempo en segundos.

Ejemplo: Transicion a una rotacion

Efecto Ease

Se usa para manipular la velocidad de la Transicion. Su sintax es {transition-timing-function: valor;} Los valores posibles para transition timing son: ease (default, lento-acelera-desacelera) linear (constant) ease-in (lento-acelera) ease-out (rapido-desacelera) ease-in-out (lento-acelera y linear desacelera) step(valor) (etapas)

En realidad hay muchas mas. ... Por ultimo tenemos un valor especial que se llama cubir-bezier que es personalizado, se recomienda usar las paginas acontinuacion para editar eso, ademas de los otros valores posibles de transiciones de css3 que se muestran alli: http://matthewlein.com/ceaser/ http://easings.net/es y cubic-bezier.com

Ejemplo: Transicion a una rotacion con cubic-bezier

Efecto delay

El delay hace que tarde el efecto una cierta cantidad de segundos en ejecutarse, la sintax seria transition-delay, por supuesto con el propietario -webkit- o el que se use. Lo coloque en el anteior "efecto ease" pero esta como comentario. Quitar el comentario en el codigo para ver que es real

Notacion acortada o shorthand notation

Como hacer css de esta manera crea una cantidad increible de css3 lo mejor es usar un shorthand notation, la sintax para ella es: transition: duracion propiedad aceleracion delay. Por ejemplo esto
{
#identificador
-webkit-transition-property: -webkit-transform;
-webkit-transition-duration: 3s;
-webkit-transition-timing-function: cubic-bezier(.57,-1.15,.05,1.83);
-webkit-transition-delay: 500ms;
}
Se convertiria en:
{#identificador: -webkit-transition: 2s -webkit-transform cubic-bezier(.57,-1.15,.05,1.83) 500ms;}

Animaciones

bases de animaciones

Para hacer una animacion en css se hace con la sintax de ejemplo donde mover es el nombre de la animacion, puede ser cualquier nombre:
@-webkit-keyframes mover {
from {transform:traslateX(-100px)}
to{transform:traslateX(600px)}
}
Luego se hace un llamado a el de esta manera:
#botonAnimacion{
-webkit-animation-name: mover;
-webkit-animation-duration: 3s;
}
Algunos otros comandos de utilidad son:
-webkit-animation-timing-function: ease; tipo de animacion...
-webkit-animation-delay: 2s ;unciona igual que el otro delay
-webkit-animation-iteration-count: 1; por defecto es uno, repite una animacion una cantidad de veces, si quiero que sea para siempre, infinite si quiero media animacion .5
-webkit-direction: normal ; se usa para especifiar la direccion de la animacion incluyendo en reversa, su timing tambien esta en reversa, los valores son normal, que es pr defecto, reverse que es alrevez, tambien esta alternate, esto permite en ciclos normal y reversa, aternate-reverse, es lo mismo pero empieza en reversa

El shorthand Notation es asi:
animation: nombre duracion timing delay iteration direction. Ej
-webkit-animation: mover 3s ease 1s 4 alternate-reverse;
Ademas se puede hacer animaciones multiples con la sintax asi:
@-webkit-keyframes mover {
from {transform:traslateX(-100px) transform:rotate(0deg)}
to{transform:traslateX(600px) transform:rotate(180deg}

Tambien se pueden hacer animaciones por porcentajes asi
@-webkit-keyframes mover {
0% {transform:traslateX(-100px)}
20% .... 100%{transform:traslateX(600px)}
}
Para finalizar una animacion se puede pausar sea cual sea con los comandos
-webkit-animation-play-state: running/paused; Con uno de esos dos valores :)
